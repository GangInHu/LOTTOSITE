\# 🍀 Lucky 645 - 로또 웹사이트



Django + Docker 기반 6/45 로또 웹사이트 과제 프로젝트



\## 기술 스택

\- Backend: Django 4.2 + Gunicorn

\- Database: PostgreSQL 15

\- Web Server: Nginx

\- Container: Docker + Docker Compose



\## 실행 방법

```bash

docker-compose up --build -d

```

브라우저에서 http://localhost 접속



\## 계정

\- 관리자: admin / admin1234

\- 일반 사용자: /register/ 에서 회원가입



\## 기능

\- 일반 사용자: 복권 구매(수동/자동), 당첨 확인

\- 관리자: 추첨 실행, 판매 내역, 당첨 내역 확인
